# Gear CMS
Currently heavily WIP.

## Requirements

*  PHP Version >= 5.6
*  PDO (MySQL 5.5, ...)

## Development

* `npm install` Install dependencies, compile js, vue & scss
* `npm run dev` Watch js & vue files
* `gulp watch` Watch scss files
